# Email Spam Detection Model

## Objective

The objective of this project is to understand how machine learning can classify emails as Spam or Not Spam.

## Tools and Technologies

* Python
* Pandas
* Scikit-learn

## Dataset

The dataset contains email messages labeled as Spam or Not Spam.

## Tasks Performed

1. Created and loaded the email dataset.
2. Converted text messages into numerical features using CountVectorizer.
3. Split the dataset into training and testing sets.
4. Trained a Naive Bayes classification model.
5. Evaluated the model accuracy.
6. Predicted whether new messages were Spam or Not Spam.

## Expected Outcome

The model successfully learns patterns in email messages and predicts whether a message is spam.

## Conclusion

This project demonstrates a basic Natural Language Processing (NLP) application using machine learning. It shows how text data can be converted into numerical features and classified using the Naive Bayes algorithm.
